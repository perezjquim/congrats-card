# Postcard generator

### Features
- Several themes (birthday, wedding, valentine's)
- Moveable textboxes
- Ability to use custom backgrounds
- Image generator (png)
- Alternative configuration via URL
	- Image width/height
	- Textboxes' position (x and y)
